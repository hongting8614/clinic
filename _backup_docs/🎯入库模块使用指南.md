## 1. 新建入库单

### 步骤1：打开新建页面

**方式一：从首页**
- 首页 → 点击"入库管理"卡片 → 点击右下角"+"按钮

**方式二：从入库列表**
- 入库列表页 → 点击右下角紫色圆形"+"按钮

### 步骤2：填写基本信息

```
┌─────────────────────────────┐
│ 基本信息                     │
├─────────────────────────────┤
│ 入库单号：RK20251105001      │  ← 自动生成
│ 入库时间：2025-11-05 14:30   │  ← 自动填充
│ 操作人：张三                 │  ← 当前用户
│ 供应商：[选择] [输入]        │  ← 可选填写
└─────────────────────────────┘
```

**供应商选择**:
- 直接输入供应商名称，或
- 点击"选择"按钮：
  - 从常用供应商快速选择
  - 搜索全部供应商
  - 新建供应商（填写名称、联系人、电话）

### 步骤3：添加药品

三种方式任选：

#### 方式一：📷 扫码入库
```
点击"📷 扫码入库" → 扫描药品条形码 → 自动添加
```
- 药品信息自动填充
- 支持连续扫码（开启后可连续添加多个药品）
- 未找到药品时可直接跳转创建

#### 方式二：📋 从档案选择
```
点击"📋 从档案选择" → 搜索或浏览 → 点击选择
```
- 支持按名称/拼音搜索
- 显示药品详细信息
- 可按分类筛选

#### 方式三：✍️ 新建药品
```
点击"✍️ 新建药品" → 填写药品信息 → 保存并自动添加
```
- 适用于新药品入库
- 同时创建药品档案

### 步骤4：填写药品明细

每个药品需要填写：

| 字段 | 说明 | 必填 | 验证规则 |
|------|------|------|----------|
| 批号 | 药品批次号 | ✅ | 不能为空 |
| 生产日期 | 生产日期 | ✅ | 不能晚于今天 |
| 有效期 | 药品有效期 | ✅ | 必须晚于生产日期；不能已过期 |
| 数量 | 入库数量 | ✅ | 必须为正整数；不超过999999 |
| 单价 | 采购单价 | ❌ | 如填写需≥0且≤999999.99 |

**填写技巧**:
- 点击"选择"按钮可从库存批次中选择（自动填充批号、日期）
- 日期选择器：点击日期字段弹出日历
- 有效期天数自动计算并显示
- 过期药品会红色标注

**表格操作**:
- 单行删除：点击行末"删除"按钮
- 批量清空：工具栏"清空明细"按钮（仅有数据时显示）

### 步骤5：操作员签名

```
滚动到底部 → 点击"点击全屏签名" → 手写签名 → 确认
```

**签名提示**:
- 首次使用会提示"建议横屏签名"（正常提示，点击"知道了"）
- 可以清空重新签名
- 签名后会显示预览

### 步骤6：提交

两种选择：

**保存草稿**
```
点击"保存草稿" → 保存成功
```
- 不需要签名
- 可稍后继续编辑
- 不会更新库存

**提交复核**
```
点击"提交复核" → 验证通过 → 提交成功 → 返回列表
```
- 必须先签名
- 所有必填项必须完整
- 提交后进入"待复核"状态
- 等待复核员复核后才会更新库存

## 2. 编辑入库单

### 可编辑的单据类型

- ✅ 草稿单据
- ✅ 已驳回单据
- ❌ 待复核单据（不可编辑）
- ❌ 已完成单据（不可编辑）

### 编辑步骤

**步骤1：打开编辑页面**
```
入库列表 → 找到草稿或驳回的单据 → 点击"编辑"或"重新编辑"
```

**步骤2：修改内容**
- 所有数据已自动加载
- 可以修改任何字段
- 可以添加/删除药品
- 可以重新签名

**步骤3：保存**
- 保存草稿：继续保持草稿状态
- 提交复核：重新提交审核

**页面标识**:
- 页面标题显示"编辑入库单"
- 单号不可修改（保持原单号）

## 3. 复核入库单

### 复核权限

**可以复核**:
- ✅ 不是自己创建的待复核单据

**不能复核**:
- ❌ 自己创建的单据（防止自我复核）
- ❌ 草稿、已完成、已驳回的单据

### 复核步骤

**步骤1：打开复核页面**
```
入库列表 → 待复核标签 → 点击单据 → 或点击"去复核"按钮
```

**步骤2：查看详情**
- 检查基本信息
- 检查药品明细
- 查看操作员签名

**步骤3：复核操作**

**通过复核**:
```
1. 在复核操作区签名
2. 点击"✓ 通过复核"
3. 确认弹窗
4. 复核完成，库存自动更新
```

**驳回**:
```
1. 填写驳回原因（必填）
2. 点击"✕ 驳回"
3. 确认弹窗
4. 驳回完成，单据返回给操作员修改
```

**复核后状态**:
- 通过：单据状态变为"已完成"，库存已更新
- 驳回：单据状态变为"已驳回"，操作员可重新编辑

## 4. 查看统计数据

### 统计面板

位置：入库列表页顶部

```
┌────────────────────────────────────┐
│        入库统计                     │
│     2025年11月05日                  │
├─────────────┬──────────────────────┤
│   今日入库   │      本周入库        │
│     5       │        23           │
│   单据数     │      单据数          │
├─────────────┼──────────────────────┤
│  本月入库    │      待复核          │
│    87       │        3            │
│   单据数     │      单据数          │
└─────────────┴──────────────────────┘
```

**统计指标说明**:

1. **今日入库**
   - 今天已完成的入库单数量
   - 统计时间：今天00:00:00至当前时刻

2. **本周入库**
   - 本周已完成的入库单数量
   - 统计时间：本周一00:00:00至当前时刻

3. **本月入库**
   - 本月已完成的入库单数量
   - 统计时间：本月1号00:00:00至当前时刻

4. **待复核**
   - 当前待复核的入库单数量
   - 实时数据

**刷新数据**:
- 下拉列表页刷新
- 统计数据自动更新

## 5. 常见问题

### Q1: 扫码后提示"未找到药品"？

**原因**: 该条码在药品档案中不存在

**解决**:
```
弹窗提示 → 点击"确定" → 自动跳转到新建药品页 → 条码已预填
→ 补充其他信息 → 保存 → 自动返回并添加
```

### Q2: 提示"药品已过期，不能入库"？

**原因**: 填写的有效期已经过期

**解决**:
- 检查有效期是否填写正确
- 过期药品确实不应入库
- 如需特殊处理，请联系管理员

### Q3: 保存草稿和提交复核有什么区别？

| 操作 | 是否需要签名 | 是否更新库存 | 后续操作 |
|------|-------------|-------------|---------|
| 保存草稿 | ❌ 不需要 | ❌ 不更新 | 可继续编辑 |
| 提交复核 | ✅ 必须 | ⏳ 复核后更新 | 等待复核 |

**建议**:
- 临时保存、暂时中断：选择"保存草稿"
- 完整填写、确认无误：选择"提交复核"

### Q4: 为什么我不能复核这个单据？

**可能原因**:
1. 这是你自己创建的单据
   - 系统禁止自我复核
   - 需要其他人复核

2. 单据不是待复核状态
   - 草稿需要先提交
   - 已完成的不需要复核

### Q5: 如何快速添加多个药品？

**方法一：连续扫码**
```
1. 开启"连续扫码"开关
2. 点击"📷 扫码入库"
3. 扫描第一个药品 → 自动添加
4. 自动再次打开扫码 → 扫描第二个
5. 重复直到全部添加完成
6. 取消扫码结束
```

**方法二：批量导入**（规划中）
- Excel导入功能
- 后续版本提供

### Q6: 签名提示"建议横屏签名"是错误吗？

**不是错误！** 这是正常的用户提示。

**说明**:
- 横屏签名空间更大，体验更好
- 提示仅首次显示
- 点击"知道了"继续
- 竖屏也可以正常签名

### Q7: 如何选择供应商？

**三种方式**:

1. **直接输入**
   - 在供应商输入框直接输入

2. **从列表选择**
   - 点击"选择"按钮
   - 常用供应商区域快速选择
   - 或搜索全部供应商

3. **新建供应商**
   - 点击"选择"→"新建供应商"
   - 填写名称、联系人、电话
   - 保存后自动选中

### Q8: 统计面板显示的数据不对？

**可能原因**:
- 数据未刷新

**解决方法**:
```
下拉列表页 → 触发刷新 → 统计数据更新
```

### Q9: 批号可以重复吗？

**可以**，但不建议。

**说明**:
- 同一药品、同一批号在同一位置（陆园/水园）会合并库存
- 建议每次入库使用唯一批号
- 相同批号会累加数量

### Q10: 删除入库单会影响库存吗？

**视情况而定**:

| 单据状态 | 是否可删除 | 是否影响库存 |
|---------|-----------|-------------|
| 草稿 | ✅ 可以 | ❌ 无影响（未入库） |
| 待复核 | ❌ 不可以 | - |
| 已完成 | ❌ 不可以 | - |
| 已驳回 | ❌ 不可以 | - |

**重要**:
- 只有草稿可以删除
- 已完成的单据不可删除（已更新库存）
- 如需撤销已完成的入库，请联系管理员
