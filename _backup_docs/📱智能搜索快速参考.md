# 📱 药品智能搜索 - 快速参考

## ⚡ 一句话说明
**输入药品名称或拼音首字母 → 选择药品 → 下拉列表自动消失 → 药品默认折叠**

---

## 🎯 使用流程

### 3步完成
```
1️⃣ 输入搜索
   ↓
2️⃣ 点击选择
   ↓
3️⃣ 自动隐藏 ✅
```

## 🔍 搜索支持

| 方式 | 示例 | 说明 |
|-----|------|------|
| **中文** | `阿` | 匹配"阿莫西林"、"阿司匹林" |
| **拼音** | `amoxilin` | 匹配"阿莫西林" |
| **首字母** | `amxl` | 匹配"阿莫西林" |
| **代码** | `AMX001` | 匹配对应药品代码 |
