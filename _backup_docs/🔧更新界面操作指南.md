# 🔧 更新界面操作指南

## 📋 问题：界面没有显示新功能

如果修改了代码但界面没有更新，请按照以下步骤操作：

---

## ✅ 操作步骤

### 步骤1：保存所有文件
```
在编辑器中：
- 按 Ctrl+S 保存当前文件
- 或 Ctrl+K, S 保存所有文件
```

### 步骤2：在微信开发者工具中重新编译

**方法1：点击编译按钮**
```
1. 找到顶部的「编译」按钮
2. 点击「编译」
3. 等待编译完成（右下角会显示进度）
```

**方法2：使用快捷键**
```
按 Ctrl+B（Windows）
或 Command+B（Mac）
```

### 步骤3：清除缓存（如果还是没有更新）

**方法1：清除工具缓存**
```
1. 点击顶部菜单「工具」
2. 选择「清除缓存」
3. 勾选「清除文件缓存」
4. 点击「清除」
5. 重新编译
```

**方法2：删除缓存目录**
```
关闭微信开发者工具
→ 删除项目下的 .cache 文件夹
→ 重新打开项目
→ 重新编译
```

### 步骤4：刷新模拟器

**方法1：点击刷新按钮**
```
在模拟器顶部找到刷新图标（🔄）
点击刷新
```

**方法2：重启模拟器**
```
1. 点击顶部的「调试器」选项
2. 选择「重启」
```

## 🆘 常见问题

### Q1: 编译后还是看不到？
**A:** 尝试完全关闭微信开发者工具，重新打开项目

### Q2: 控制台报错？
**A:** 截图控制台错误信息，检查是否有语法错误

### Q3: 页面是空白的？
**A:** 检查是否选择了正确的页面路径

### Q4: 显示"构建输出"有错误？
**A:** 查看具体错误信息，可能是代码语法问题
