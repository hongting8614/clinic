# ⚡ 入库快速参考

## 🚀 3秒开始入库

```
首页 → 📦 新建入库单 → 开始入库
```

---

## 📝 5步完成入库

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1️⃣ | 点击首页紫色横幅 | 📦 新建入库单 |
| 2️⃣ | 扫码/搜索添加药材 | 📷 或 🔍 |
| 3️⃣ | 填写批号、日期、数量 | 必填项 ⭐ |
| 4️⃣ | 操作员签名 | 必填 ✅ |
| 5️⃣ | 提交入库单 | 完成 🎉 |

---

## ✅ 必填项检查清单

- [ ] 至少1个药材
- [ ] 批号
- [ ] 生产日期
- [ ] 有效期
- [ ] 数量 > 0
- [ ] 操作员签名

---

## 💡 快捷操作

| 功能 | 操作 |
|------|------|
| 扫码添加 | 点击 📷 |
| 搜索添加 | 输入药材名 |
| 删除药材 | 点击 ✕ |
| 签名 | 点击签名框 |
| 保存草稿 | 点击保存草稿 |
| 提交 | 点击提交复核 |

---

## ⚠️ 重要提示

1. **新药材在最上方** - 最新添加的显示在列表顶部
2. **复核人签名可选** - 不是必填项
3. **自动计算效期** - 填写有效期后自动显示剩余天数
4. **过期药材不能入库** - 系统会阻止

---
