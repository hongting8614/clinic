# ⚡ 双人入库快速参考

## 🔄 两个阶段

```
阶段1: 操作员录入 → 操作员签名 → 提交待复核
阶段2: 复核人复核 → 复核人签名 → 通过入库
```

---

## 👤 操作员操作（5步）

| 步骤 | 操作 |
|------|------|
| 1️⃣ | 首页点击 📦 新建入库单 |
| 2️⃣ | 扫码/搜索添加药品 |
| 3️⃣ | 填写批号、日期、数量 |
| 4️⃣ | 操作员签名 ✍️ |
| 5️⃣ | 点击"提交复核" |

**结果**: 状态变为"待复核"

---

## 👨‍⚕️ 复核人操作（5步）

| 步骤 | 操作 |
|------|------|
| 1️⃣ | 入库管理 → 待复核标签 |
| 2️⃣ | 点击"去复核"按钮 |
| 3️⃣ | 查看入库单明细 |
| 4️⃣ | 复核人签名 ✍️ |
| 5️⃣ | 点击"通过并入库" |

**结果**: 库存更新，状态变为"已完成"

---

## 📱 快速入口

### 操作员
```
首页 → 📦 新建入库单
```

### 复核人
```
首页 → 入库管理 → 待复核 → 去复核
```

---

## ✅ 必填项

### 操作员
- [ ] 至少1个药品
- [ ] 批号、日期、数量
- [ ] 操作员签名

### 复核人
- [ ] 复核人签名
- [ ] 选择通过/驳回

---

## 🎯 关键点

| 要点 | 说明 |
|------|------|
| 双人操作 | 操作员 ≠ 复核人 |
| 独立页面 | 录入和复核分开 |
| 双签名 | 操作员签名 + 复核人签名 |
| 可驳回 | 复核人可驳回并说明原因 |
| 状态流转 | 待复核 → 已完成/已驳回 |

---

## 📊 状态说明

| 状态 | 说明 | 下一步 |
|------|------|--------|
| 待复核 | 等待复核人审批 | 复核人操作 |
| 已完成 | 已入库 | 查看详情 |
| 已驳回 | 需要修改 | 操作员修改 |

---

## ⚠️ 注意事项

### 操作员
- ✅ 确保信息准确
- ✅ 必须签名
- ❌ 不能自己复核

### 复核人
- ✅ 仔细核对
- ✅ 必须签名
- ✅ 驳回要说明原因
- ❌ 通过后不可撤销

---

## 🆘 常见问题

| 问题 | 解决方案 |
|------|---------|
| 找不到待复核单据 | 检查"待复核"标签 |
| 无法签名 | 检查签名权限 |
| 驳回后怎么办 | 操作员修改后重新提交 |
| 通过后发现错误 | 联系管理员处理 |
