## 🎯 使用方式

### 方式一：首页快捷入口（新增）⭐
1. 打开小程序首页
2. 在"快捷操作"区域找到 **"📦 新建入库单"** 横幅
3. 点击即可直接进入新建入库单页面
4. 开始扫码或搜索添加药材

**特点**：
- ✅ 一键直达
- ✅ 醒目易找
- ✅ 操作便捷

### 方式二：入库管理（原有）
1. 打开小程序首页
2. 点击"入库管理"卡片
3. 进入入库单列表
4. 点击右下角"+"按钮新建
