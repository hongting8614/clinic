## ⚙️ 云函数环境变量配置

### 位置
```
微信开发者工具 → 云开发 → 云开发控制台 → 云函数 → drugBarcodeQuery → 配置 → 环境变量
```

### 变量配置
```
变量名：JISUAPI_APPKEY
变量值：[粘贴您获取的AppKey]
```

## 🚀 部署步骤（3步完成）

```bash
1. 添加环境变量 → 保存
2. 右键云函数 → 云端安装依赖（不上传node_modules）
3. 右键云函数 → 上传并部署：云端安装依赖
```
