# 用药信息界面字号统一规范

## 📐 统一后的字号体系

### 核心原则
- **视觉层次清晰**：重要信息大，次要信息小
- **字号差异合理**：相邻级别差值 4-8rpx
- **整体协调统一**：所有元素符合统一标准

---

| 元素 | 优化前 | 优化后 | 说明 |
|------|--------|--------|------|
| **药材图标** | 64rpx | 56rpx | ⬇️ 减小，更协调 |
| **药材名称** | 40rpx | 36rpx | ⬇️ 减小，保持醒目 |
| **药材规格** | 26rpx | 24rpx | ⬇️ 统一次要信息字号 |
| **标签图标** | 28rpx | 26rpx | ⬇️ 统一图标大小 |

| **单位显示** | 36rpx | 32rpx | ⬇️ 统一单位字号 |
| **提示文字** | 24rpx | 22rpx | ⬇️ 统一提示字号 |
| **批次信息** | 26rpx | 24rpx | ⬇️ 统一信息字号 |

## 📊 字号层级结构

### 一级 - 核心信息（最大）
```
44rpx - 数量输入框（最重要的用户输入）
36rpx - 药材名称（最重要的信息）
```

### 二级 - 辅助信息（中等）
```
32rpx - 单位显示
28rpx - 标题/标签
```

### 三级 - 补充信息（较小）
```
24rpx - 规格/标签文字/批次信息
22rpx - 提示文字
```

### 四级 - 装饰元素（图标）
```
56rpx - 药材图标
26rpx - 标签图标
```

### 优化后的优势

## 🎯 设计原则

### 1. **黄金比例**
```
大字号 : 中字号 : 小字号 = 1.6 : 1.3 : 1.0
44 : 32 : 24 ≈ 1.8 : 1.3 : 1.0
```

### 2. **最小差值**
```
相邻层级字号差值 ≥ 4rpx
避免 26rpx 和 28rpx 同时出现在相邻位置
```

### 3. **统一标准**
```
标题类：28-36rpx
正文类：24rpx
提示类：22rpx
```

### 4. **可读性优先**
```
最小字号：22rpx（提示文字）
主要内容：≥ 24rpx
用户输入：≥ 44rpx
```
