# 签名框高度优化说明

## 📋 优化目标

将签名框（未全屏状态）的高度减半，使界面更紧凑，节省屏幕空间。

---

## 🔧 修改内容

### 1. 签名框容器高度

**修改前**：`120rpx`  
**修改后**：`60rpx`

```scss
.signature-container {
  height: 60rpx; // 从 120rpx 减半
}
```

---

### 2. 占位符图标大小

**修改前**：`40rpx`  
**修改后**：`24rpx`

```scss
.placeholder-icon {
  font-size: 24rpx; // 从 40rpx 缩小
}
```

---

### 3. 占位符文字大小

**修改前**：`24rpx`  
**修改后**：`20rpx`

```scss
.placeholder-text {
  font-size: 20rpx; // 从 24rpx 缩小
}
```

---

### 4. 图标与文字间距

**修改前**：`gap: 12rpx`  
**修改后**：`gap: 6rpx`

```scss
.signature-placeholder {
  gap: 6rpx; // 从 12rpx 减半
}
```

---

### 5. "重新签名" 按钮

#### 按钮内边距
**修改前**：`padding: 8rpx 20rpx`  
**修改后**：`padding: 4rpx 16rpx`

#### 按钮字体
**修改前**：`font-size: 24rpx`  
**修改后**：`font-size: 20rpx`

#### 按钮圆角
**修改前**：`border-radius: 20rpx`  
**修改后**：`border-radius: 16rpx`

```scss
.btn-resign {
  padding: 4rpx 16rpx;
  font-size: 20rpx;
  border-radius: 16rpx;
}
```

---

### 6. 按钮位置

**修改前**：
```scss
bottom: 20rpx;
right: 20rpx;
```

**修改后**：
```scss
bottom: 8rpx;
right: 12rpx;
```

## 🎯 设计原则

1. **等比例缩小**：所有元素按比例缩小，保持视觉和谐
2. **留白减少**：缩小间距和内边距，提高空间利用率
3. **可点击性保持**：虽然高度减半，但仍保持足够的点击区域（60rpx ≈ 30px）
4. **全屏签名不变**：全屏签名弹窗的尺寸保持不变，确保签名体验
