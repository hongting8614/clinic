| 字段 | 支持语音 | 示例 |

## 💡 使用技巧

### **✅ DO（推荐做法）：**

1. **说话清晰**
   - 慢速、清楚地说
   - 普通话发音

2. **环境安静**
   - 在安静的诊室使用
   - 关闭背景音乐

3. **识别后检查**
   - 尤其是规格字段
   - 手动修正错误

4. **长药材名称用语音**
   - "盐酸左氧氟沙星片"
   - 比打字快很多

---

### **❌ DON'T（避免）：**

1. **说话太快**
   - 含糊不清
   - 识别率低

2. **嘈杂环境**
   - 多人说话
   - 背景噪音大

3. **方言口音重**
   - 影响识别
   - 建议用键盘

4. **短药材名称**
   - "碘伏"、"棉签"
   - 直接打字更快

## 🆘 常见问题

### **Q：点击🎤没反应？**

→ 微信 → 设置 → 隐私 → 授权管理 → 开启麦克风

### **Q：识别不准？**

→ 手动修正为"0.3g×32粒/盒"
