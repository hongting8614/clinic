## 💡 使用技巧

### **技巧1：使用标准普通话**
```
✅ 标准发音识别率高
⚠️ 方言可能影响识别
```

### **技巧2：环境安静**
```
✅ 安静环境识别准确
❌ 嘈杂环境识别率低
```

### **技巧3：说话清晰**
```
✅ 慢速清晰：布-洛-芬-缓-释-胶-囊
❌ 快速含糊：布洛芬...
```

### **技巧4：识别后检查**
```
规格字段特别注意：
- 数字和单位
- 特殊符号（×、/）
- 手动修正（如需要）
```

## 🆘 常见问题

### **Q1：找不到🎤按钮？**

**A：** 检查输入法设置
```
方法1：切换输入法
- 长按输入框
- 选择其他输入法
- 微信键盘、搜狗等都有语音

方法2：更新输入法
- 确保输入法是最新版本
- 在设置中开启语音输入
```

---

### **Q2：识别不准确？**

**A：** 优化说话方式
```
1. 说话慢一点、清楚一点
2. 环境安静一点
3. 使用标准普通话
4. 识别后手动修正
```

---

### **Q3：规格识别错误？**

**A：** 手动修正即可
```
说："零点三克乘三十二粒每盒"
可能识别成："0.3克×32粒/盒"
手动改成："0.3g×32粒/盒" ✅

这是正常的，语音只是辅助
最终需要检查确认
```

---

### **Q4：为什么不直接集成语音识别？**

**A：** 输入法更好用
```
原因：
1. 小程序没有语音识别API
2. 输入法识别准确率更高
3. 用户已经习惯使用
4. 无需额外开发和维护
5. 完全免费稳定

结论：
使用输入法语音是最佳方案 ✅
```
