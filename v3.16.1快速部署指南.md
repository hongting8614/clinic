# ⚡ v3.16.1 快速部署指南

> 🎯 **5分钟快速部署流程**

---

## 🚀 快速部署（3步完成）

### 第1步：编译测试（1分钟）

```
1. 打开微信开发者工具
2. 点击"编译"按钮
3. 检查是否有错误
```

✅ **检查点**：
- 编译成功，无红色错误
- 模拟器正常显示页面

---

### 第2步：快速验证（2分钟）

在模拟器中测试：

**测试A：诊断症状一致性**
```
1. 进入"门诊登记"
2. 输入主诉："崴脚"
3. 选择诊断："踝关节扭伤"
4. ✅ 检查症状是否显示："脚踝肿胀；疼痛；活动受限"
```

**测试B：修改诊断**
```
1. 继续上面的操作
2. 修改诊断为："腕关节扭伤"
3. ✅ 检查症状是否自动更新为："手腕肿胀；疼痛；活动受限"
```

---

### 第3步：上传代码（2分钟）

```
1. 点击工具栏"上传"按钮
2. 填写版本信息：
   版本号：v3.16.1
   备注：修复处方库存提示和诊断症状一致性问题
3. 点击"上传"
4. ✅ 等待上传完成
```

---
