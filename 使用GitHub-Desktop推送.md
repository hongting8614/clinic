# 使用 GitHub Desktop 推送代码（更简单）

如果命令行认证遇到问题，使用 GitHub Desktop 会更方便！

## 第一步：下载并安装 GitHub Desktop

1. 访问：https://desktop.github.com/
2. 下载 Windows 版本
3. 安装（一路下一步）

## 第二步：登录 GitHub

1. 打开 GitHub Desktop
2. 点击：Sign in to GitHub.com
3. 在浏览器中完成登录
4. 授权 GitHub Desktop

## 第三步：添加现有仓库

1. 点击：File → Add local repository...
2. 选择路径：D:\AK-PMS
3. 点击：Add repository

## 第四步：推送代码

1. GitHub Desktop 会自动识别已提交的内容
2. 点击顶部：Publish repository（发布仓库）
3. 或点击：Push origin（推送）

✅ 完成！代码自动上传到 GitHub！

## 优点

- ✅ 无需配置 Token
- ✅ 图形化界面，更直观
- ✅ 自动处理认证
- ✅ 可视化查看修改
- ✅ 一键推送和拉取

## 日常使用

**推送代码：**
1. 在 GitHub Desktop 中看到修改
2. 填写提交信息
3. 点击：Commit to master
4. 点击：Push origin

**拉取代码：**
1. 点击：Fetch origin
2. 如果有更新，点击：Pull origin

非常简单！🚀
