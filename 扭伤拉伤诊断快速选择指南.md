## 📖 每个诊断的详细说明

### 1. 踝关节扭伤（最常见）

**适用情况**：
- 崴脚、扭到脚踝
- 脚踝疼痛、肿胀
- 行走困难

**典型主诉**：
- "崴脚后脚踝疼痛肿胀"
- "脚踝扭伤后活动受限"
- "踩空后脚踝疼痛"

**自动填充内容**：
```
疾病名称：扭伤
症状：脚踝肿胀；疼痛；活动受限
处置：冷敷；包扎固定；制动休息；抬高患肢
推荐用药：云南白药气雾剂、红花油
```

## 🔍 扭伤 vs 拉伤的区别

### 扭伤（Sprain）

**定义**：关节扭转导致韧带损伤

**特点**：
- 发生在关节部位
- 通常有明显的扭转动作
- 关节肿胀、疼痛

**诊断列表中的扭伤**：
- ✅ 踝关节扭伤
- ✅ 腕关节扭伤
- ✅ 膝关节扭伤
- ✅ 肩关节扭伤

**处置**：冷敷、包扎、制动

---

### 拉伤（Strain）

**定义**：肌肉或肌腱过度拉伸导致损伤

**特点**：
- 发生在肌肉部位
- 通常是过度用力或突然动作
- 肌肉疼痛、僵硬

**诊断列表中的拉伤**：
- ✅ 颈部肌肉拉伤
- ✅ 腰部肌筋膜拉伤
- ✅ 外侧副韧带拉伤

**处置**：热敷（24小时后）、按摩、休息

## 🎯 常见问题解答

### Q1：如何区分踝关节扭伤和外侧副韧带拉伤？

**A**：
- **踝关节扭伤**：整个脚踝扭伤，疼痛在脚踝周围
- **外侧副韧带拉伤**：特指膝盖外侧韧带损伤，疼痛在膝盖外侧

**简单判断**：
```
脚踝受伤  →  踝关节扭伤
膝盖受伤  →  膝关节扭伤 或 外侧副韧带拉伤
```

### Q2：患者说"扭伤了"，但不知道具体部位怎么办？

**A**：询问患者：
```
┌─────────────────────────────────────┐
│ 医生："哪里扭伤了？"                 │
│                                      │
│ 患者："脚踝"  →  选择：踝关节扭伤    │
│ 患者："手腕"  →  选择：腕关节扭伤    │
│ 患者："膝盖"  →  选择：膝关节扭伤    │
│ 患者："肩膀"  →  选择：肩关节扭伤    │
│ 患者："脖子"  →  选择：颈部肌肉拉伤  │
│ 患者："腰"    →  选择：腰部肌筋膜拉伤│
└─────────────────────────────────────┘
```

### Q3：可以选择多个诊断吗？

**A**：可以！如果患者同时多处受伤：
```
示例：患者摔倒，同时扭伤脚踝和手腕

诊断框输入：
踝关节扭伤；腕关节扭伤

（用分号分隔多个诊断）
```

### Q4：选错了诊断怎么办？

**A**：
1. **提交前**：直接修改诊断框的内容
2. **提交后**：可以编辑记录，修改诊断

### Q5：这7个诊断够用吗？

**A**：
- ✅ 覆盖了90%以上的常见扭伤/拉伤情况
- ✅ 如果遇到特殊情况，可以手动输入其他诊断
- ✅ 系统支持自定义诊断

## 📝 总结

### 核心要点

1. **7个诊断，覆盖所有常见情况**
   - 4个关节扭伤（踝、腕、膝、肩）

   - 游客崴脚最常见
